Cấu trúc dữ liệu trong JavaScript

Ngày đăng: 29/12/2025

JavaScript cung cấp nhiều cấu trúc dữ liệu mạnh mẽ để lưu trữ và quản lý dữ liệu. Trong bài này, chúng ta sẽ tìm hiểu về Array, Object và Map – ba cấu trúc phổ biến nhất.

1. Array (Mảng)

Array dùng để lưu trữ danh sách các phần tử có thứ tự.

let fruits = ["Táo", "Chuối", "Cam"];
console.log(fruits[0]); // Output: Táo

fruits.push("Xoài");
console.log(fruits); // ["Táo", "Chuối", "Cam", "Xoài"]

2. Object (Đối tượng)

let person = {
    name: "Nguyễn Văn A",
    age: 25,
    job: "Lập trình viên"
};

console.log(person.name); // Nguyễn Văn A
person.age = 26;

3. Map

let myMap = new Map();
myMap.set("name", "Nguyễn Văn A");
myMap.set(1, "Số một");

console.log(myMap.get("name")); // Nguyễn Văn A
