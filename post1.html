Giới thiệu về Java cơ bản

Ngày đăng: 29/12/2025

Java là một trong những ngôn ngữ lập trình phổ biến nhất thế giới, được phát triển bởi Sun Microsystems (nay thuộc Oracle) từ năm 1995. Java nổi tiếng với nguyên tắc “Write Once, Run Anywhere” nhờ JVM (Java Virtual Machine).

Cài đặt môi trường Java

1. Tải JDK từ trang oracle.com hoặc adoptium.net
2. Cài đặt và thiết lập biến môi trường JAVA_HOME
3. Kiểm tra bằng lệnh: java -version

Chương trình Hello World đầu tiên

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Xin chào thế giới Java!");
    }
}

Biên dịch: javac HelloWorld.java
Chạy: java HelloWorld
