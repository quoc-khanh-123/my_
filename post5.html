Xử lý exception trong Java

Ngày đăng: 29/12/2025

Java cung cấp cơ chế try-catch-finally để xử lý exception an toàn.

Cú pháp try-catch

try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("Lỗi chia cho 0: " + e.getMessage());
} finally {
    System.out.println("Khối finally luôn chạy");
}

Lợi ích: Chương trình không crash, người dùng nhận thông báo thân thiện.
