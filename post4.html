Async/Await trong JavaScript

Ngày đăng: 29/12/2025

Async/Await là cú pháp hiện đại giúp xử lý bất đồng bộ dễ đọc hơn, thay thế cho Promise chain.

Cú pháp

async function fetchData() {
    try {
        let response = await fetch('https://api.example.com/data');
        let data = await response.json();
        console.log(data);
    } catch (error) {
        console.error('Lỗi:', error);
    }
}

fetchData();

Async/Await làm code trông giống đồng bộ, dễ xử lý lỗi với try-catch.
