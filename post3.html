Lập trình hàm trong Java (Lambda Expression)

Ngày đăng: 29/12/2025

Lambda Expression được giới thiệu từ Java 8, giúp viết code ngắn gọn hơn khi làm việc với Functional Interface.

Cú pháp cơ bản

Runnable r = () -> System.out.println("Hello Lambda!");
new Thread(r).start();

Ví dụ với List và Stream

List names = Arrays.asList("An", "Bình", "Cường");

names.forEach(s -> System.out.println(s));

Lợi ích: Code ngắn gọn, dễ đọc, hỗ trợ lập trình hàm mạnh mẽ.
