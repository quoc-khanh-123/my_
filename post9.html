Spring Boot cơ bản với Java

Ngày đăng: 29/12/2025

Spring Boot giúp tạo ứng dụng Java nhanh chóng, đặc biệt là REST API.

Tạo REST Controller

@RestController
@RequestMapping("/api")
public class UserController {

    @GetMapping("/hello")
    public String hello() {
        return "Xin chào Spring Boot!";
    }
}

Chạy ứng dụng → truy cập http://localhost:8080/api/hello

Ưu điểm: Tự động cấu hình, ít boilerplate code.
