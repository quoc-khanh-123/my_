Multithreading trong Java

Ngày đăng: 29/12/2025

Multithreading cho phép chạy nhiều luồng đồng thời.

Cách implement Runnable

class MyRunnable implements Runnable {
    public void run() {
        System.out.println("Luồng đang chạy");
    }
}

Thread t = new Thread(new MyRunnable());
t.start();

Lợi ích: Tăng tốc độ xử lý, phù hợp ứng dụng cần chạy song song.
